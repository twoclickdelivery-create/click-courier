# Assets

Сюда положи:

- `icon.png` — 1024×1024 PNG, иконка приложения
- `adaptive-icon.png` — 1024×1024 PNG, foreground для Android adaptive icon
- `splash.png` — 1284×2778 PNG, splash screen (можно меньше)
- `favicon.png` — 48×48 PNG, для web
- `notification-icon.png` — 96×96 PNG, белая (only alpha) для Android уведомлений

Пока их нет — Expo CLI создаст дефолтные при первом запуске `npx expo start`.

Простой способ сгенерировать иконки из логотипа: https://romannurik.github.io/AndroidAssetStudio/icons-launcher.html
или https://www.appicon.co/
